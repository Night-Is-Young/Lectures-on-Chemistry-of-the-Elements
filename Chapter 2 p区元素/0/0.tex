\section{0族元素综述}
\subsection{0族元素的发现与命名}
我们从元素的希腊文开始介绍\footnote{笔者并不是要求读者掌握希腊文的意思,只是作为繁忙的学习中的一点调剂.}.

稀有气体单质由于其反应惰性(\ce{Ar}就得名于此),因此直到1785年才被发现.首个发现的即~\ce{Ne}~(它也得名于此),此后其它的稀有气体单质才陆续地被发现.\\
\indent 在很长的一段时间内,人们都认为由于其满壳层的电子结构带来的稳定性使得它们不参与化学反应,因此很长一段时间内它们都被称作\tbf{惰性气体},然而后面的研究推翻了这一点.因此称它们为\tbf{稀有气体}更加合适,因为空气中的稀有气体单质含量较低.然而,一些稀有气体元素(尤其是~\ce{Ar},来源于地壳中大量存在的~\ce{^{40}K}的$\beta$衰变)跟稀土一样在地壳中绝对含量并不算少,因此严谨来讲``稀有''也是历史上的相对概念.
\subsection{0族元素的物理性质}
所有稀有气体单质都是无色无味的无毒气体.由于这些物质在组成分子上的相似性,因此其物理性质呈现很好的递变规律.

原子间的范德华力随着极化率的增大而增大,这在蒸发焓和沸点上有着直接的体现.\\
\indent 稀有气体原子的电离能在同周期是最高的.然而,较重的稀有气体元素的电离能却小于~\ce{O}和~\ce{F}等元素,这也暗示了它们的化合物的形成.我们将在后面重点提及这些化合物.\\
\indent 作为这一族中不寻常的一个,~\ce{He}(准确而言是~\ce{^4He})在接近$0\K$时有着神奇的物相变化.在大约$2.2\K$时,~\ce{He}会从正常的液体(\ce{He(I)}相)转变为超流体\footnote{由于~\ce{^4He}是自旋为零的Bose子,因此会发生Bose-Einstein凝聚.这涉及到二级相变的内容,并不要求掌握.}(\ce{He(II)}相).超流体~\ce{^4He}具有异常高的热导率和比热容,其动力粘度为零.这一流体最值得称道的性质是其超流性.~\ce{He(II)}会沿着置于其中的空杯的杯壁爬升并越过杯口进入容器中,直到内外液面相平.
\section{0族元素的分布和应用}
\subsection{\ce{He}}
\paragraph{\ce{He}的分布}
宇宙中的~\ce{He}是第二丰富的元素.然而由于其过低的密度,因此不能被地球的重力场所维系,因此地球上的~\ce{He}较少,主要的是~\ce{^4He},来源于各种重元素的$\alpha$衰变.\\
\indent 然而,月球上却有着远比地球多得多的~\ce{^3He}储备.这些~\ce{^3He}主要来源于太阳风,估计有约100万吨的~\ce{^3He}吸附在月球表面的硅酸盐矿物的晶格孔隙中.而地球由于磁场的保护,太阳风并不能长驱直入地击中地表,因此地球上的~\ce{^3He}较为少见.
\paragraph{\ce{He}的用途}
\indent \ce{He}的最大的用途是作为各种工业或实验过程的保护气.例如作为加压和吹扫气体,电弧焊接时的保护气体,及参与制造芯片的化学反应过程.日常生活中的小用途则是作为气球或飞艇上升所需的气体.在密度较小而安全的气体中几乎只有这一选择(充~\ce{H2}的气艇在历史上发生过不止一起爆炸事件.)\\
\indent 液氦由于其较低的沸点($-268.93\tc$)可以用于冷却剂,研究各种超导物质.\\
\indent 由于氦在血液中的溶解度很低,因此可以加到氧气中防止减压病,作为潜水员的呼吸用气体,或用于治疗气喘和窒息.\\
\indent 与密度和空气密度不同的任何气体一样,吸入少量氦气会暂时改变人声的音调.在理想气体中,声速$c$满足
\[c=\sqrt{\dfrac{\gamma p}{\rho}}=\sqrt{\dfrac{\gamma RT}{M}}\]
其中$\gamma$是气体的绝热指数, $M$是气体的分子量.人的声道本质上是一个共振腔,其共振频率$f$与声速$v$成正比.对于分子量更小,密度更低的氦气,人在其中发出的声音频率更高,听起来也就更尖锐.
\subsection{其余0族元素}
其余0族元素的主要应用是霓虹灯中的保护气.尽管它们确实显现不同的颜色,但一般产生鲜艳的颜色仍然需要额外加入荧光物质.\\
\indent 除去产量极低的~\ce{Ne},~\ce{Ke}和~\ce{Xe}以外,~\ce{Ar}也常作为保护气使用.
\section{\ce{Xe}的化学性质}
\subsection{\ce{Xe}的首个化合物}
自N. Barlett发现~\ce{Xe}的电离能小于~\ce{O2},受到启发而制备得到~\ce{XePtF6}以来,稀有气体化合物的族谱在化学史上徐徐展开.将~\ce{PtF6}蒸汽与等量的~\ce{Xe}混合即可得到橙黄色固体\ce{XePtF6},制备反应的方程式可以写作:
\begin{tightcenter}
    \ce{Xe + PtF6 -> XePtF6}
\end{tightcenter}
\indent 研究表明\cite{GRAHAM2000321},~\ce{XePtF6}实际上并非~\ce{[Xe]+[PtF6]-},而含有~\ce{[XeF]+[PtF6]-},~\ce{[XeF]+[Pt2F11]-},~\ce{[Xe2F3]+[PtF6]-}等化合物.这一反应并不像开始看起来那样简单,而且计量关系也很可能比较复杂.\\
\indent \ce{XePtF6}具有强氧化性,遇水迅速分解:
\begin{tightcenter}
    \ce{2XePtF6 + 6H2O -> 2Xe + O2 + 2PtO2 + 12HF}
\end{tightcenter}
在此之后,人们又制备出了许多~\ce{Xe(MF6)_x}形式的化合物,并最终制备得到了~\ce{Xe}的氟化物.
\subsection{\ce{Xe}的氟化物}
\paragraph{\ce{Xe-F}平衡体系}
\ce{Xe}和~\ce{F2}在加热下仅生成~\ce{XeF2},~\ce{XeF4}和~\ce{XeF6}.这也是~\ce{Xe}仅有的三种氟化物.它们总是处于平衡之中:
\begin{tightcenter}
    \ce{Xe + F2 <=> XeF2}\ \ \ $K_1(774.2\K)=29.8$\\
    \ce{Xe + 2F2 <=> XeF4}\ \ \ $K_2(774.2\K)=0.50$\\
    \ce{Xe + 3F2 <=> XeF6}\ \ \ $K_3(774.2\K)=3.3\times10^{-3}$
\end{tightcenter}
这一体系有时也用于平衡计算的练习.总之,控制温度,投料比等条件可以选择性的合成比较纯的~\ce{XeF_n}$(n=2,4,6)$.

需要注意的是,~\ce{XeF6}是变形八面体结构的.早期的研究表明$C_{2\text{v}}$或$C_{3\text{v}}$对称性的结构与光谱数据相对更加符合.因此,人们提出了~\ce{XeF6}中具有空间效应的孤对电子,它所处的位置(棱心/面心)将导致其余键发生轻微的畸变并形成具有上述对称性的结构:

显然,传统的VSEPR模型不能对~\ce{XeF6}的构型给出一个令人满意的解释.一种比较合理的解释是采用定性的分子轨道理论.我们可以由~\ce{Xe}的$5\text{s}$, $5\text{p}$和$6$个~\ce{F}原子的$2\text{p}$轨道构造八面体分子轨道,如下图所示:
\begin{figure}[H]\centering
    \resizebox{0.7\textwidth}{!}{
    \input{figure/XeF6-MO.tex}}
    \caption{\ce{XeF6}的定性分子轨道组合图}
\end{figure}
相比于正八面体的~\ce{SF6},~\ce{XeF6}多了一对电子,使得HOMO轨道变为$2\text{a}_{1\text{g}}$, LUMO轨道变为$2\text{t}_{1\text{u}}$.两者继续组合可以使得$2\text{a}_{1\text{g}}$的能量进一步下降得到稳定化作用.根据群论的知识,要使得轨道发生组合,振动模式的不可约表示必须包括
\[\text{A}_{1\text{g}}\otimes\text{T}_{1\text{u}}=\text{T}_{1\text{u}}\]
查阅$O_{\text{h}}$点群的特征标表,可知$\text{T}_{1\text{u}}$对应于三重简并的振动,它在$x$, $y$和$z$三个方向上等价,因此将使得~\ce{XeF6}分子的对称性降低至$C_{3\text{v}}$.
\paragraph{\ce{XeF2}}
将~\ce{Xe}和~\ce{F2}混合后光照,或者将~\ce{F2}与过量的~\ce{Xe}混合后在~\ce{Ni}制容器中加热即可得到~\ce{XeF2}.
\begin{substance}[\ce{XeF2}]
    二氟化氙,化学式为~\ce{XeF2},是无色固体,室温下易升华.气态~\ce{XeF2}具有令人作呕的恶臭味.
\end{substance}
固体~\ce{XeF2}具有简明的晶体结构:
\chemfig{XeF2}{0.125}{\ce{XeF2}的晶体结构\quad$\text{I}\,4/\text{m}\,\text{m}\,\text{m}$\quad$a=b=\SI{422.5}{\pico\meter},c=\SI{695.3}{\pico\meter}$}
同其它氟化氙相比,~\ce{XeF2}是稍温和一些的氟化剂和氧化剂.因此,它经常被用作氟化剂.\\
\indent \ce{XeF2}也可以与某些酸(主要是各非金属的最高价含氧酸或衍生物)反应生成~\ce{Xe(OR)2}.例如:
\begin{tightcenter}
    \ce{XeF2 + HOSO2F -> FXe(OSO2F) + HF}\\
    \ce{FXe(OSO2F) + HOSO2F -> Xe(OSO2F)2 + HF}
\end{tightcenter}
\ce{Xe(OSO2F)}和~\ce{FXe(OSO2F)}对热不稳定,容易分解:
\begin{tightcenter}
    \ce{Xe(OSO2F)2 ->T[$\Delta$] Xe + S2O6F2}\\
    \ce{2FXe(OSO2F) ->T[$\Delta$] Xe + XeF2 + S2O6F2}
\end{tightcenter}

\indent 高溴酸及其盐的制备一直以来是一个难题.最初人们采取核反应的方式由~\ce{SeO4^2-}制得~\ce{BrO4-}:
\begin{tightcenter}
    \ce{^83SeO4^2- -> ^83BrO4- + e-}
\end{tightcenter}
后来,人们首次使用~\ce{XeF2}氧化~\ce{BrO3^-}制得了\ce{BrO4-}:
\begin{tightcenter}
    \ce{BrO3- + XeF2 + H2O -> BrO4- + Xe + 2HF}
\end{tightcenter}

\indent \ce{XeF2}的水解也是值得探究的一个问题.总体的方程式如下:
\begin{tightcenter}
    \ce{2XeF2 + 2H2O -> 2Xe + O2 + 4HF}
\end{tightcenter}
这一反应的机理可以表示如下:
\begin{tightcenter}
    \ce{XeF2 + H2O -> XeO + 2HF}\\
    \ce{XeO + H2O -> Xe + H2O2}\\
    \ce{H2O2 + XeF2 -> Xe + O2 + 2HF}
\end{tightcenter}
在碱或金属离子的催化下这一反应将被大大加速.
\paragraph{\ce{XeF4}}
将~\ce{Xe}与~\ce{F2}以1:5的比例混合后在~\ce{Ni}制容器中加热即可得到~\ce{XeF4}.
\begin{substance}[\ce{XeF4}]
    四氟化氙,化学式为~\ce{XeF4},是无色固体,室温下易升华.
\end{substance}
\ce{XeF4}的晶体结构如下:
\chemfig{XeF4}{0.125}{\ce{XeF4}的晶体结构\quad$\text{P}\,1\,21/\text{n}\,1$\quad$a=\SI{501.5}{\pico\meter},b=\SI{573.9}{\pico\meter},c=\SI{590.4}{\pico\meter},\beta=100.86^\circ$}
\ce{XeF4}的水解是比较复杂的.总反应方程式可以表示如下:
\begin{tightcenter}
    \ce{6XeF4 + 12H2O -> 2XeO3 + 4Xe + 3O2 + 24HF}
\end{tightcenter}
反应的过程大致可以表示如下:
\begin{tightcenter}
    \ce{3Xe^{IV} ->T[\ce{H2O}] 2Xe^{II} + Xe^{VIII}}\\
    \ce{Xe^{II} ->T[\ce{H2O}] Xe + O2}\\
    \ce{Xe^{VIII} ->T[\ce{H3O+}] Xe^{VI} + O2}
\end{tightcenter}
我们将在讲到~\ce{XeO3}时再次提及这一反应.\\
\indent 和~\ce{XeF2}一样,~\ce{XeF4}也可以作为氟化剂,但氧化性更强.
\paragraph{\ce{XeF6}}
将~\ce{Xe}与~\ce{F2}以1:20的比例混合后在~\ce{Ni}制容器中加热即可得到~\ce{XeF4}.此外,用~\ce{O2F2}氧化\ce{XeF4}也是可行的办法.
\begin{substance}[\ce{XeF6}]
    六氟化氙,化学式为\ce{XeF6},是无色固体,室温下易升华,其蒸汽显现黄色.
\end{substance}

\indent 尽管~\ce{XeF6}也可以氟化其他物质,但由于过于猛烈的氧化性而总是导致底物的裂解,因此很少用它作氟化剂.
\subsection{\ce{Xe}的氧化物和含氧阴离子}
\paragraph{\ce{XeO3}与\ce{HXeO4-}}
前面已经说过,~\ce{XeO3}可以由~\ce{XeF4}或~\ce{XeF6}的水解得到,但最好还是用~\ce{XeF6}为宜.
\begin{substance}[\ce{XeO3}]
    三氧化氙,化学式为~\ce{XeO3},是一种无色透明的吸湿性晶体.
\end{substance}
\ce{XeO3}的晶体结构如下:
\chemfig{XeO3}{0.125}{\ce{XeO3}的晶体结构\quad$\text{P}\,2_1\,2_1\,2_1$\quad$a=\SI{616.3}{\pico\meter},b=\SI{811.5}{\pico\meter},c=\SI{523.4}{\pico\meter}$}
\ce{XeO3}具有猛烈的爆炸性.稍有摩擦或撞击,或在潮湿的环境中就容易发生爆炸(同时伴随着蓝色的闪光):
\begin{tightcenter}
    \ce{2XeO3 -> 2Xe + 3O2}
\end{tightcenter}
在碱性溶液中,~\ce{Xe^{VI}}主要以~\ce{HXeO4-}的形式存在,并更容易发生分解.
\paragraph{\ce{XeO6^4-}与~\ce{XeO4}}
前面说到~\ce{HXeO4-}在碱性条件下容易发生分解,事实上发生了如下的歧化反应:
\begin{tightcenter}
    \ce{2HXeO4- + 2OH- -> XeO6^4- + Xe + O2 + H2O}
\end{tightcenter}
因此,~\ce{XeF6}在碱性条件下的水解也能获得类似的结果:
\begin{tightcenter}
    \ce{2XeF6 + 16OH- -> XeO6^4- + Xe + O2 + 12F- + 8H2O}
\end{tightcenter}
将~\ce{Ba2XeO6}加入浓~\ce{H2SO4}中即可制得~\ce{XeO4}.这是一种极不稳定的四面体形分子,在熔融时即发生爆轰.
\paragraph{\ce{XeO2}}
和前面两种氧化物相比,~\ce{XeO2}则发现的相对较晚\cite{Brock2011}.由于~\ce{XeF4}水解时很容易发生歧化,因此需要在$\SI{0}{\celsius}$下,浓度为$\SI{2}{\mole\per\liter}$的~\ce{H2SO4}中对~\ce{XeF4}水解得到橙黄色固体~\ce{XeO2}.尽管使用VSEPR预测~\ce{XeO2}是折线形分子,但实际上在固体中~\ce{Xe}被四个~\ce{O}原子配位,形成平面四方形的单元(与~\ce{XeF4}类似)并与其它单元连接.
\paragraph{\ce{Xe3O2}与~\ce{Xe2O5}}在高压下可以合成具有更加复杂计量比的~\ce{Xe3O2}与~\ce{Xe2O5}\cite{Dewaele2016}.两者的晶体结构如下所示:
\chemfig{Xe3O2}{0.125}{\ce{Xe3O2}的晶体结构\quad$\text{I}\,\text{m}\,\text{m}\,\text{m}$\quad$a=\SI{316.6}{\pico\meter},b=\SI{409.4}{\pico\meter},c=\SI{845.7}{\pico\meter}$}

\section{其它0族元素的化学性质}
几十年来,所有0族元素都已经制备得到了化合物(尽管大部分都很不稳定).例如,~\ce{Kr}能形成与~\ce{XeF2}结构类似的~\ce{KrF2}.这些物质的性质与结构可以参考~\ce{Xe}的类似化合物.这里我们介绍一类独特的物质.
\subsection{高压下的稀有气体阴离子}
稀有气体元素的闭壳层结构阻止了将电子填入其价轨道.事实上,在体相中制备或在气相中检测到的含稀有气体的阴离子中,稀有气体元素总是处于零或正氧化态.后来,人们发现在高达几百GPa的压力下工作,可以制备出含有带负电~\ce{Ng}(代表稀有气体元素)原子的化合物.\\
\indent 在高于$\SI{125}{\giga\pascal}$的压力下可以观察到~\ce{MeXe}和~\ce{Mg2Xe}的生成.压力继续升高至$\SI{250}{\giga\pascal}$后预测也可以生成~\ce{Kr}和~\ce{Ar}的类似物.在这类物质中,除去~\ce{Xe}的周围外,相邻的~\ce{Mg}之间也有明显的电子密度较高的区域\cite{Miao2015},这表明晶格空间中存在量子化的轨道并被电子占据,形成间隙准原子(Interstitial Quastiatom, ISQ).
\chemfig{Mg2Xe-ELF}{0.2}{\ce{Mg2Xe}沿$[100]$晶面的电子局域化函数(Electronic Localization Function, ELF)图}
\indent 这类化合物中比较著名的是~\ce{Na2He}.这也是首次合成的~\ce{He}的化合物,由两种单质在金刚石对顶砧中以超过$\SI{160}{\giga\pascal}$的压力合成.~\ce{Na2He}具有典型的反萤石型结构,~\ce{Na}原子做简单立方堆积,~\ce{He}原子填入一半的立方体空隙中.电子密度图显示另一半空置的立方体空隙也具有较高的电子密度,这说明~\ce{Na2He}也是一种电子化合物,其组成写作~\ce{(Na+)2(e^-)2He}更为合适.在~\ce{Na}和~\ce{He}上的电荷分别约为$+0.6\,\e$和$-0.15\,\e$,而形式上的$2\e^-$中心的电荷约为$-1.1\,\e$.理论计算也预测了~\ce{Li2He}的存在,但更重的碱金属形成的~\ce{K2He},~\ce{Rb2He}和~\ce{Cs2He}在$\SI{1000}{\giga\pascal}$下都不能稳定存在.这可能是较重的碱金属在高压下发生了$n\text{s}$电子到$(n-1)\text{d}$电子的跃迁,变得更加局域化而倾向于不给出电子导致的.

    \[p(x,t)=P(x)\cos(\omega t)\]

    \[\dfrac{\d^2 P}{\d x^2}+k^2P=0\]
