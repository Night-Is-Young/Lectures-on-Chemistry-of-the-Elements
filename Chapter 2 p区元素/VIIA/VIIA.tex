\chapter{VIIA族元素}
\section{VIIA族元素综述}
VIIA族元素,即我们常说的\tbf{卤素},其研究历史由来已久,在化学中也有十分重要的地位. 1811年Schweigger研究氯的性质时提出了卤素这一概念,意味产生盐的元素.在汉语中,卤的本意是制盐时剩余的黑色母液.
\subsection{VIIA族元素的发现与分布}
\paragraph{氟}
在17世纪,人们就发现了具有腐蚀性的氢氟酸~\ce{HF}和具有荧光性质的萤石~\ce{CaF2}. 1812年, Ampere提出了氟元素的概念,其名字即来源于萤石.然而,受限于当时实验室的条件和安全措施,直到1886年氟单质才由Moissan制备得到. Moissan采用的办法是用~\ce{Pt/Ir}电极在~\ce{Pt}管内电解~\ce{KHF2}的无水~\ce{HF}溶液:
\begin{tightcenter}
    \ce{2KHF2 ->[\text{电解}] 2KF + H2 + F2}
\end{tightcenter}
\indent 卤素单质的活泼性质决定它们在自然界中几乎总是以游离态存在.地壳中~\ce{F}的丰度为第13位,与~\ce{Mn},~\ce{Ba}相近.主要的存在形式是:萤石~\ce{CaF2},冰晶石~\ce{Na3AlF6}和氟磷灰石~\ce{Ca5(PO4)3F}.其中只有萤石被广泛用于合成氟及相关的化合物;自然界中的冰晶石是一种稀有的矿物,仅在格陵兰岛存在有开采价值的矿床,工业中电解铝所需要的冰晶石是人工合成的;氟磷灰石则主要用于制备磷酸盐.
\paragraph{氯}
氯是卤素中最早被分离出的.1774年, Scheele用~\ce{MnO2}氧化~\ce{HCl}得到~\ce{Cl2}单质:
\begin{tightcenter}
    \ce{MnO2 + 4HCl ->[\Delta] MnCl2 + 2H2O + Cl2}
\end{tightcenter}
受燃素理论影响,他并没有意识到自己制备得到了一种新的单质.直到后来人们才认识到这种黄绿色的物质是~\ce{Cl2}单质.1811年, Davy根据其颜色将其命名为氯.\\
\indent 氯在地壳中的储存不多,主要以岩盐~\ce{NaCl}的形式存在.大部分氯元素以~\ce{Cl-}的形式在海水或内陆咸水湖中存在.
\paragraph{溴}
有关溴元素的早期应用之一可以追溯到罗马时期使用的贵重的紫色颜料骨螺紫,实际上是$6,6'$-二溴靛蓝. 1826年, Balard从蒙彼利埃的咸水湖中提取得到了棕黄色的溴单质.由于它具有刺激性且令人不适的气味,因此将该元素命名为溴(希腊语$\beta\rho\omega\mu o\zeta$,发恶臭的).\\
\indent 溴在地壳中的含量比氟和氯都少.和氯一样,溴元素主要在海洋中以~\ce{Br-}的形式存在.
\paragraph{碘}
1811年, Courtios首次制备单质碘.当时正值拿破仑战争,因此需要大量~\ce{Na2CO3}开采硝石.前者可以在海藻灰中提取,剩余的废物则用硫酸后处理并销毁.在一次销毁过程中,他加入了过量的硫酸,并观察到紫色的蒸气生成.这些蒸气会遇冷凝结为具有紫黑色金属光泽的单质.囿于经费不足,他将样品寄送给他的朋友, 最终交给了Gay Lussac. 1813年, Gay Lussac确认这种物质是一种新元素的单质,并根据单质的颜色将这种元素命名为碘(希腊语$i\omega\delta\eta\zeta$,深紫罗兰色的).碘的中文名称取自英文名称的最后一个音节dine,加上代表固体非金属元素的石字旁,得名为碘.\\
\indent 碘在地壳中的含量比更轻的几种卤素都小,和一些不常见的金属~\ce{Tl},~\ce{Tm}相当.由于~\ce{I-}较强的还原性,碘很少以碘化物矿的形式存在,主要是碘酸盐矿物,例如碘钙石~\ce{Ca(IO3)2}.大部分碘也存在于海洋等水体中,但含量也比氯或溴少.一些海藻能够富集碘元素.
\paragraph{砹}
1940年,来自UC Berkeley的研究人员用回旋加速器合成了第85号元素,并根据其强烈的不稳定性用希腊文$\alpha\sigma\tau\alpha\tau o\zeta$(意为``不稳定'')将其命名为astatine,中文命名为砹.\\
\indent 不久后的1943年,~\ce{At}元素作为~\ce{^{235}_{92}U}衰变链上的痕量中间产物被发现存在于自然界中.~\ce{At}是地壳中丰度最低的天然元素,估计在任何时候地壳中~\ce{At}元素的总量不超过$\SI{1}{\gram}$\cite{Hollerman2001}.

\paragraph{单质的颜色}
从\autoref{table:卤素的物理性质}~可以看出,随着原子序数的增加,卤素单质的颜色逐渐变深.根据分子轨道理论,卤素分子的HOMO轨道是$\pi^\ast$轨道, LUMO轨道是$\sigma^\ast$轨道.采用库普曼斯定理近似,则分子的电离能的负值近似等于HOMO轨道能量;类似地,分子的电子亲和能的负值近似等于LUMO轨道能量.以下是各~\ce{X2}的电离能数据以及近似得到的$\text{HOMO}-\text{LUMO}$能量差:

\indent 碘单质在溶液相中的颜色取决于所选溶剂的性质.在非极性溶剂(例如~\ce{CCl4},脂肪烃)中,~\ce{I2}与气相中的颜色相同,都为紫色;在芳香烃中~\ce{I2}的颜色是粉红色或浅棕色.在极性溶剂(例如~\ce{H2O},~\ce{EtOH}等)中,~\ce{I2}分子与溶剂分子形成电荷转移配合物,此时发生的跃迁变为由溶剂分子的HOMO轨道(通常是孤对电子)到~\ce{I2}分子的$\sigma^\ast$轨道的跃迁,呈现出棕色.因此,在应用碘量法测定时如果向体系内加入~\ce{CCl4},~\ce{I2}将被萃取到有机相中并显紫色,而判断反应结束的现象则是有机相中的紫色消失.

\subsection{单质的化学性质}
\paragraph{\ce{F2}}在所有单质中,~\ce{F2}是最活泼的,也是氧化性最强的.在适当条件下,~\ce{F2}能与除~\ce{He},~\ce{Ne}和~\ce{Ar}以外的所有元素反应生成化合物,并且通常氧化到最高或较高的价态.块状的~\ce{Al},~\ce{Fe},~\ce{Ni}和~\ce{Cu}等金属元素单质与~\ce{F2}反应后可以在表面生成一层氟化物保护膜,阻止继续反应.因此,通常用~\ce{Ni}制容器作为~\ce{F2}参与反应时的反应器.
\paragraph{\ce{Cl2},~\ce{Br2}和~\ce{I2}}
较重的卤素的反应活性明显低于~\ce{F2},但仍然比较活泼,反应活性按$\ce{Cl2}>\ce{Br2}>\ce{I2}$的顺序递减.这些单质参与的反应将在介绍有关的物质时提到.
\subsection{单质的制备方法}
\paragraph{\ce{F2}}
直至今日,大规模制备~\ce{F2}单质仍然采用Moissan的方法,即电解~\ce{KHF2}与~\ce{HF}的混合物,只是电解的温度维持在$\SI{100}{\celsius}$左右.
\paragraph{\ce{Cl2}}
工业上大部分~\ce{Cl2}来源于氯碱工业,即电解~\ce{NaCl}溶液(通常直接由海水浓缩处理得到,因此大部分氯碱厂都设在海边):
\begin{tightcenter}
    \ce{2NaCl + 2H2O ->[\text{电解}] Cl2 + H2 + 2NaOH}
\end{tightcenter}
实验室中是被~\ce{Cl2}仍可以采用加热~\ce{MnO2}和~浓~\ce{HCl}的办法.
\paragraph{\ce{Br2}}
溴的工业制备主要是向含有~\ce{Br-}的酸性卤水中通入~\ce{Cl2}得到~\ce{Br2}单质:
\begin{tightcenter}
    \ce{2Br- + Cl2 -> 2Cl- + Br2}
\end{tightcenter}
生成的~\ce{Br2}挥发性较强,可以通入空气或水蒸气将~\ce{Br2}从液相转移到气相(这一步操作称作\tbf{吹出}).随后经过进一步纯化即得到~\ce{Br2}单质.
\paragraph{\ce{I2}}
碘的工业制备通常有两种方法.一种是将含有~\ce{I-}的海水用~\ce{Cl2}氧化并提纯得到~\ce{I2},与制备~\ce{Br2}相似:
\begin{tightcenter}
    \ce{2I- + Cl2 -> 2Cl- + I2}
\end{tightcenter}
另一种办法是从含有~\ce{IO3-}的智利硝石中提取~\ce{I2}.首先萃取浓缩得到~\ce{NaIO3}母液,然后加入~\ce{NaHSO3}将其还原为~\ce{I-}:
\begin{tightcenter}
    \ce{2IO3- + 6HSO3- -> 2I- + 6SO4^{2-} + 6H+}
\end{tightcenter}
向生成的酸性溶液中加入母液,即可得到~\ce{I2}单质:
\begin{tightcenter}
    \ce{IO3- + 5I- + 6H+ -> 3I2 + 3H2O}
\end{tightcenter}

\subsection{制备和用途}
\paragraph{\ce{HF}}
无水~\ce{HF}由浓硫酸与萤石反应得到:
\begin{tightcenter}
    \ce{CaF2 + H2SO4(conc.) -> CaSO4 + 2HF}
\end{tightcenter}
所用的萤石必须具有较高的纯度,尤其不能含有硅酸盐.否则~\ce{Si^{IV}}转化为~\ce{H2SiF6}将消耗$6$当量的~\ce{HF}造成大量损失.\\
\indent \ce{HF}的主要用途是制备冰晶石用于铝的冶炼;提纯~\ce{U}元素;生产各类含氟化合物,例如聚四氟乙烯等.\\
\indent \ce{HF}及其水溶液具有强烈的腐蚀性和毒性.~\ce{HF}灼伤的显著特征是出现白色伤痕,剧痛以及肌肉痉挛.前者是由于高浓度~\ce{HF}导致的局部失水.除此之外,~\ce{HF}的特殊毒性来源于~\ce{F-}.它与细胞中的~\ce{Ca^{2+}}结合形成难溶的~\ce{CaF2},导致~\ce{K+}过量而带来剧痛;同时,~\ce{Ca^{2+}}是维持肌肉正常收缩的重要离子,因此急性~\ce{F-}中毒将导致肌肉痉挛抽搐和心脏停搏.\\
\indent 由于上述原因,使用~\ce{HF}或含有~\ce{F-}的物质时必须小心.如果不慎接触,可以使用注射葡萄糖酸钙解毒,通过短时间摄入大量~\ce{Ca^{2+}}与~\ce{F-}结合减轻毒性.
\paragraph{\ce{HCl}}
工业生产~\ce{HCl}已经有相当长的一段历史,采用的办法主要由以下几种:
\begin{enumerate}[label=\tbf{\roman*.},leftmargin=*]
    \item \tbf{Mannheim法}\\
    \indent 该法是历史上第一次大规模制备~\ce{HCl}的方法.作为Leblanc制碱法中的一步\cite{Plessen2000},该法利用~\ce{HCl}的挥发性,用~\ce{NaCl}与浓~\ce{H2SO4}的反应制备~\ce{HCl(g)}:
    \begin{tightcenter}
        \ce{NaCl + H2SO4 ->[\Delta] NaHSO4 + HCl}\\
        \ce{NaCl + NaHSO4 ->[\Delta] Na2SO4 + HCl}
    \end{tightcenter}
    \item \tbf{Hargreaves法}\\
    \indent 作为Mannheim法的衍生方法,该法采用~\ce{SO2},~\ce{H2O}和空气的混合物代替~\ce{H2SO4}进行反应:
    \begin{tightcenter}
        \ce{4NaCl + 2SO2 + O2 + 2H2O ->[\Delta] 2Na2SO4 + 4HCl}
    \end{tightcenter}
    前述两种方法至今在岩盐资源丰富或对~\ce{Na2SO4}有大量需求的地方仍有应用.
    \item \tbf{直接合成法}\\
    \indent 现代氯碱工业已经可以直接生产~\ce{H2}和~\ce{Cl2}.在石英炉中点燃两者即可得到~\ce{HCl}:
    \begin{tightcenter}
        \ce{H2 + Cl2 -> 2HCl}
    \end{tightcenter}
    \item \tbf{氯化反应副产物}\\
    \indent 如今大部分~\ce{HCl}都来源于对各种有机物的氯化反应产生的副产物,例如氯乙烯的生产:
    \begin{tightcenter}
        \ce{CH2=CH2 + Cl2 -> ClHC=CH2 + HCl}
    \end{tightcenter}
\end{enumerate}
\indent ~\ce{HCl}在工业上的用途比较简单,大部分时候用于提供酸性环境,进行酸洗或中和等,因此用量并不大.如今,随着对环境保护的要求越来越高,处理多余的~\ce{HCl}已经成为比较主要的问题,一些时候纯度不高的~\ce{HCl}的价格甚至是负数.
\paragraph{\ce{HBr}和~\ce{HI}}
浓硫酸与溴化物和碘化物的类似反应会导致~\ce{HBr}和~\ce{HI}部分被氧化为~\ce{Br2}或~\ce{I2}:
\begin{tightcenter}
    \ce{2HX + H2SO4 -> X2 + 2H2O + SO2}\quad$\ce{X}=\ce{Br},\,\ce{I}$
\end{tightcenter}
换用浓~\ce{H3PO4}可以取得比较好的效果.也可以通过红磷与~\ce{X2}的反应原位制备~\ce{PX3},随后水解得到对应的~\ce{HX}:
\begin{tightcenter}
    \ce{2P + 3X2 -> 2PX3}\quad$\ce{X}=\ce{Br},\,\ce{I}$\\
    \ce{PX3 + 3H2O -> H3PO3 + 3HX}\quad$\ce{X}=\ce{Br},\,\ce{I}$
\end{tightcenter}
需要注意的是,如果用~\ce{Br2}进行反应,~\ce{H3PO3}会被继续氧化为~\ce{H3PO4}:
\begin{tightcenter}
    \ce{H3PO3 + H2O + Br2 -> 2HBr + H3PO4}
\end{tightcenter}
\ce{I2}与联氨水溶液的定量反应也可以用于制备~\ce{HI}或~\ce{HBr}:
\begin{tightcenter}
    \ce{2I2 + N2H4 ->[H2O] 4HI + N2}
\end{tightcenter}

\indent \ce{HBr}和~\ce{HI}主要用于制备溴化物和碘化物,除此之外应用较少.
\subsection{~\ce{HX}的化学性质}
\paragraph{\ce{HX}的水溶液化学}
所有~\ce{HX}都在水中发生电离:
\begin{tightcenter}
    \ce{HX + H2O <=> X- + H3O+}\quad$K_a(\ce{X})=\dfrac{\con{X-}\con{H3O+}}{\con{HX}}$
\end{tightcenter}
其解离平衡常数$K_a$及相关热力学数据如下表所示:
\begin{table}[H]\centering
    \caption{卤化氢在水中的电离平衡常数及相关热力学数据$(T=\SI{298}{\kelvin})$}

    \begin{tabular}{lcccc}
    \noalign{\global\arrayrulewidth=1pt}\hline\noalign{\global\arrayrulewidth=0.4pt}
    & \ce{HF} & \ce{HCl} & \ce{HBr} & \ce{HI} \\

从\autoref{table:卤化氢的酸性}~可以看出,~\ce{HF}的酸性也是反常的,在所有~\ce{HX}中只有~\ce{HF}是弱酸. Pauling认为是~\ce{F}的高电负性决定了水相中分子态~\ce{HF}相较解离后的离子的稳定性\cite{Pauling1960}.需要注意的是,~\ce{HF}在水中难解离与其较高的键能没有直接关系,因为电离是生成阴阳离子,而键断裂是生成两个中性的原子.\\
\indent 最近的研究提出了另一种解释\cite{Ayotte2005}.首先,根据化学反应热力学基本原理有
\[\ln K_a(T)=-\dfrac{-\Delta G_{\text{r}}}{RT}=\dfrac{1}{R}\left(\Delta S_{\text{r}}-\dfrac{\Delta H_{\text{r}}}{T}\right)\]
从\autoref{table:卤化氢的酸性}~可以看出,除去~\ce{HF}的电离焓$\Delta H_{\text{r}}^\ominus$较小之外,其较小的电离熵$\Delta S_{\text{r}}^\ominus$也进一步抑制其电离.这意味着~\ce{F-}相比其它~\ce{X-}具有更小的水合熵.这是由于~\ce{F}的半径与~\ce{O}接近,在水中~\ce{F-}可以替代~\ce{OH-}构成氢键网络,使得周围的水分子趋向有序化;而其余~\ce{X-}会破坏水的氢键网络,水合熵明显更大.这说明熵效应也是导致~\ce{HF}是弱酸的重要因素.\\
\indent 如果水中~\ce{HF}浓度继续升高,则可以生成稳定阴离子~\ce{[HF2]^-}而酸性增强:
\begin{tightcenter}
    \ce{HF + F- -> [HF2]-}\quad$K=\dfrac{\con{HF2^-}}{\con{HF}\con{F-}}$
\end{tightcenter}
这一离子具有直线型的对称氢键结构:
\chemfig{HF2-}{1}{\ce{[HF2]^-}的结构}
此外,在低温条件下也可以制备得到含有~\ce{H_nF_{n+1}-}离子的~\ce{HF}水合物.它们的结构与固态~\ce{HF}中长链的一部分相同.
\paragraph{\ce{HX}的非水溶剂化学}
和~\ce{NH3},~\ce{H2O}一样,液态~\ce{HF}也是一种优良的非水溶剂,具有很高的电导率,介电常数和较低的粘度.液态~\ce{HF}中发生如下形式的自耦电离:
\begin{tightcenter}
    \ce{3HF -> [H2F]+ + [HF2]-}
\end{tightcenter}
很多金属氟化物都能很好地溶于~\ce{HF}并形成相应的~\ce{[MF_x]^{y-}}离子.非金属元素的氟化物(例如~\ce{XeF2}和~\ce{SF6})以及部分高价的金属氟化物(例如~\ce{MoF6},~\ce{UF6}等)也可以溶解,但不发生解离.\\
\indent 除氟化物之外的无机溶质溶于~\ce{HF}常发生溶剂解反应,生成氟代物.这也是制备无水的金属氟化物(例如~\ce{TiF4},~\ce{VF3}等)的很好的办法.典型的溶剂解反应如下:
\begin{tightcenter}
    \ce{K2SO4 + 4HF ->[$\SI{-80}{\celsius}$] 2K+ + 2HF2- + H2SO4}\\
    \ce{H2SO4 + 3HF ->[$\SI{0}{\celsius}$] H3O+ + HSO3F + HF2-}\\
\end{tightcenter}
磷酸则按下面的方式逐步溶剂解:
\begin{tightcenter}
    \ce{H3PO4 + 3HF -> H2PO3F + H3O+ + HF2-}\\
    \ce{H2PO3F + 3HF -> HPO2F2 + H3O+ + HF2-}\\
    \ce{HPO2F2 + 6HF -> 2H3O+ + PF6- + HF2-}
\end{tightcenter}
~\ce{MnO4-}和~\ce{CrO4^{2-}}也在~\ce{HF}溶剂解为相应的氟氧化物~\ce{MnO3F}和~\ce{CrO2F2}.\\
\indent 液态~\ce{HF}具有较强的酸性,可以使~\ce{H2O},~\ce{ROH},~\ce{RCOOH}质子化.此外,~\ce{HF}也可以作为Lewis碱提供~\ce{F-}.\\
\indent 相比~\ce{HF},其余的~\ce{HX}的沸点更低,容易气化,介电常数小,因此很少作为溶剂.然而,它们仍然是制备对应的含~\ce{X-}的无水化合物的重要介质.

\subsection{卤素互化物及多卤离子的化学性质}
\subsubsection{概述}
\autoref{table:卤素互化物的物理性质与制备方法}~已经给出了相关化合物的制备方法.卤素互化物的主要化学性质体现在其氧化性,以及作为Lewis酸/碱形成多卤离子的性质(由此也衍生出这些化合物作为非水溶剂的化学).大部分卤素互化物因存在正价的卤素中心而具有明显的氧化性.卤素氟化物的反应活性的次序为
\begin{tightcenter}
    $\ce{ClF3}>\ce{BrF5}>\ce{IF7}>\ce{ClF}>\ce{BrF3}>\ce{IF5}>\ce{BrF}>\ce{IF3}>\ce{IF}$
\end{tightcenter}
可以看出,对于卤素氟化物~\ce{XF_n}而言,给定$n$时反应能力的顺序为$\ce{Cl}>\ce{Br}>\ce{I}$,而给定卤素中心~\ce{X}时反应能力的次序为$\ce{XF5}>\ce{XF3}>\ce{XF}$.只有对于~\ce{ClF5}例外,它的反应性稍弱于~\ce{ClF3}.对于不含~\ce{F}的卤素互化物,反应性通常低于对应比例的氟化物.
\subsubsection{\ce{XY}的化学性质}
在所有~\ce{XY}型卤素互化物中最稳定的是~\ce{ClF}和~\ce{ICl}.~\ce{BrF}和~\ce{IF}容易发生歧化(尤其是后者):
\begin{tightcenter}
    \ce{3BrF -> Br2 + BrF3}\\
    \ce{5IF -> 2I2 + IF5}
\end{tightcenter}
在$\SI{298}{\kelvin}$下,~\ce{IBr}一定程度上分解为对应的单质,而~\ce{BrCl}则基本分解.\\
\indent \ce{XY}型卤素互化物的化学反应基本可以分为以下三类.
\paragraph{卤化反应}
和卤素单质类似,卤素互化物也可以作为卤化试剂.~\ce{ClF}是一种有效的氟化剂,可以使很多单质反应生成氟化物和~\ce{Cl2}:
\begin{tightcenter}
    \ce{W + 6ClF -> WF6 + 3Cl2}\\
    \ce{Se + 4ClF -> SeF4 + 2Cl2}
\end{tightcenter}
\ce{ClF}也可以对多重键进行加成反应,或对低价元素的化合物进行氧化加成反应:
\begin{tightcenter}
    \ce{(CF3)2C=O + ClF -> (CF3)2FC-OCl}\\
    \ce{SF4 + ClF -> SF5Cl}
\end{tightcenter}
\ce{ClF}与~\ce{-OH}或~\ce{-NH}基团反应时剧烈放热,使底物发生氯化反应并脱除~\ce{HF}:
\begin{tightcenter}
    \ce{H2O + 2ClF -> 2HF + Cl2O}\\
    \ce{HNO3 + ClF -> HF + ClONO2}\\
    \ce{HNF2 + ClF -> HF + HF2Cl}
\end{tightcenter}
\indent \ce{ICl}和~\ce{IBr}的反应虽然相比~\ce{ClF}温和一些,但仍然比较剧烈,例如对金属单质的卤化反应:
\begin{tightcenter}
    \ce{2P + 10ICl -> 2PCl5 + 5I2}\\
    \ce{2V + 6ICl -> 2VCl3 + 3I2}
\end{tightcenter}
\ce{ICl}和~\ce{IBr}与有机物的反应与条件密切相关.使用~\ce{ICl}蒸气可以将酚氯化,而在溶液中主要发生碘化作用:
\begin{tightcenter}
    \ce{ICl + PhOH ->[\text{气相}] PhOCl + HI}\\
    \ce{ICl + PhOH ->[\text{溶液相}] PhOI + HCl}
\end{tightcenter}
这说明在气相中~\ce{ICl}分解为~\ce{Cl2}和~\ce{I2},并由性质更活泼的前者与羟基发生反应.在液相中的反应可以分情况讨论:在介电常数较低的~\ce{CCl4}溶液中主要发生协同的取代反应,此时氧原子的n电子应该与$\sigma^\ast$系数更高的~\ce{I}原子端作用,最终生成碘代产物并离去~\ce{Cl-}离子\footnote{在~\ce{ICl}的异裂,生成的~\ce{I+}迅速进行碘化反应,但笔者认为在低介电常数的溶液中发生异裂并不符合常理.}.而在高介电常数的~\ce{PhNO2}溶液中,~\ce{ICl}主要发生异裂,生成活性的~\ce{I+}并迅速发生碘代反应.\\
\indent 相比~\ce{ICl},~\ce{IBr}更难以异裂为阴阳离子.~\ce{IBr}对芳香化合物的卤化反应总是生成溴代产物.这是因为~\ce{IBr}在溶液中也分解成~\ce{Br2}和~\ce{I2},并由活性更高的~\ce{Br2}进行溴代反应.
\paragraph{Lewis酸碱反应}
\ce{ClF}作为Lewis酸可以与氟离子供体(例如~\ce{NOF},碱金属氟化物等)反应生成直线型的~\ce{[ClF2]-}离子:
\begin{tightcenter}
    \ce{ClF + NOF -> [NO]+[ClF2]-}
\end{tightcenter}
\ce{ClF}也可以与更强的Lewis酸反应生成折线形的~\ce{[Cl2F]+}离子(注意原子连接方式为~\ce{[Cl-Cl-F]+}),例如:
\begin{tightcenter}
    \ce{2ClF + BF3 -> [Cl2F]+[BF4]-}\\
    \ce{2ClF + AsF5 -> [Cl2F]+[AsF6]-}
\end{tightcenter}
\paragraph{作为非水溶剂}
\indent\ce{ICl}和~\ce{IBr}熔融时都能部分离解为离子:
\begin{tightcenter}
    \ce{3IX <=> [I2X]+ + [IX2]-}\quad$\ce{X}=\ce{Cl},\,\ce{Br}$
\end{tightcenter}
因此也可以作为一种非水溶剂.例如,向~\ce{ICl}中加入Lewis酸/碱都能明显提高其电导率:
\begin{tightcenter}
    \ce{ICl + AlCl3 -> [I2Cl]+[AlCl4]-}\\
    \ce{ICl + MCl -> M+[ICl2]-}\quad\ce{C6H5N + 2ICl -> [C6H5N-I]+[ICl2]-}\\
    \ce{ICl + PCl5 -> [PCl4]+[ICl2]-}\quad\ce{2ICl + SbCl5 -> [I2Cl]+[SbCl6]-}
\end{tightcenter}
\subsubsection{\ce{XY3}的化学性质}
\paragraph{\ce{ClF3}的化学性质}
对~\ce{XY3}型化合物的研究主要集中在~\ce{ClF3}上.这是一种极强的氟化剂,能与大多数物质发生反应,包括~\ce{Pt},~\ce{Au}等贵金属单质,~\ce{Xe}等稀有气体单质,塑料等有机物,甚至是沙,混凝土等氧化物.室温下可以用低碳钢储存~\ce{ClF3}.最耐腐蚀的仍是~\ce{Ni}单质或Monel合金.纯净的~\ce{ClF3}也不会腐蚀石英或硼硅酸玻璃,但在痕量~\ce{HF}的存在下即可发生腐蚀.\\
\indent \ce{ClF3}能将大多数氯化物或氧化物转化成氟化物:
\begin{tightcenter}
    \ce{2AgCl + 2ClF3 -> 2AgF2 + Cl2 + 2ClF}\\
    \ce{2Co3O4 + 6ClF3 -> 6CoF3 + 3Cl2 + 4O2}
\end{tightcenter}
\indent 适当稀释的~\ce{ClF3}也与~\ce{NH3}或~\ce{N2H4}发生剧烈反应.与~\ce{N2H4}的反应曾经用于火箭发动机中,后来因为安全问题而被否决\footnote{John D. Clark在\textit{Ignition!: An informal history of liquid rocket propellants}中写道:~``值得注意,当这层氟化物融化或被刮伤,且无法及时再次形成时,消防员就要面对一场金属与~\ce{ClF3}反应引发的火灾.对于应对这种事故,我一直以来都建议实验人员为自己准备一双好跑鞋.''\cite{Clark1972}}.反应的方程式如下:
\begin{tightcenter}
    \ce{2NH3 + 2ClF3 -> 6HF + N2 + Cl2}\\
    \ce{3N2H4 + 4ClF3 -> 12HF + 3N2 + 2Cl2}
\end{tightcenter}
\indent \ce{ClF3}与~\ce{NH4F}或~\ce{NH4HF2}反应可以得到~\ce{NF2Cl}和~\ce{NFCl2}.反应的机理如下\cite{Miyazaki2018}:
\begin{tightcenter}
    \ce{NH4F + ClF3 -> [NH3ClF2]F + HF}\\
    \ce{[NH3ClF2]F + ClF3 -> NF2Cl + ClF + 3HF}\\
    \ce{[NH3ClF2]F + ClF -> NFCl2 + 3HF}\\
    \ce{2[NH3ClF2]F + ClF3 -> N2 + 3ClF + 6HF}
\end{tightcenter}
因此总反应方程式为
\begin{tightcenter}
    主反应:~\ce{2NH4F + 3ClF3 -> NF2Cl + NFCl2 + 8HF}\\
    副反应:~\ce{2NH4F + 3ClF3 -> N2 + 3ClF + 8HF}
\end{tightcenter}
\paragraph{\ce{BrF3}的化学性质}
从前面的反应性顺序可以看出~\ce{BrF3}比~\ce{ClF3}这样猛烈的氟化剂弱一些,但仍然可以与大部分物质发生反应.~\ce{BrF3}与氧化物反应通常定量地放出~\ce{O2}并生成相应的氟化物以及~\ce{Br2}单质,例如:
\begin{tightcenter}
    \ce{2B2O3 + 4BrF3 -> 4BF3 + 2Br2 + 3O2}\\
    \ce{3SiO2 + 4BrF3 -> 3SiF4 + 2Br2 + 3O2}
\end{tightcenter}
这类反应可以作为一种定量分析方法测定金属或合金中存在的少量~\ce{O}.\\
\indent 对于含氧酸盐,~\ce{BrF3}通常只能起到部分氟化作用,并且常常伴随着复杂的价态变化,例如:
\begin{tightcenter}
    \ce{3KReO4 + 4BrF3 -> 3KReO2F4 + 2Br2 + 3O2}\\
    \ce{6K2Cr2O7 + 16BrF3 -> 12KCrOF4 + 8Br2 + 15O2}\\
    \ce{6KClO3 + 10BrF3 -> 6KBrF4 + 4Br2 + 9O2 + 6ClO2F}
\end{tightcenter}
这些反应通常都呈现出复杂的结果,似乎没有很好的规律.\\
\indent \ce{BrF3}也可以作为非水溶剂,并且在液态时具有相当大的电导率:
\begin{tightcenter}
    \ce{2BrF3 <=> [BrF2]+ + [BrF4]-}
\end{tightcenter}
然而反常的是升温会造成~\ce{BrF3}电导率下降,这是因为在温度较高时~\ce{[BrF2]+}和~\ce{[BrF4]-}离子不稳定.在~\ce{BrF3}中发生的各种反应和前述的卤素互化物中发生的反应类似,例如:
\begin{tightcenter}
    \ce{KF + BrF3 -> K[BrF4]}\\
    \ce{Ag[BrF4] + [BrF2][AuF4] -> Ag[AuF4] + 2BrF3}
\end{tightcenter}
这类反应也被广泛用于制备含氟的化合物,因为~\ce{BrF3}既是优良的氟化剂,又可以在反应结束后很容易地通过减压除去.
\paragraph{\ce{I2Cl6}的化学性质}
和同类的化合物一样,~\ce{I2Cl6}可以作为一种强烈的氯化剂,不过也许部分原因是因为它容易分解生成~\ce{Cl2}.~\ce{I2Cl6}的水解反应比较复杂,总的反应为:
\begin{tightcenter}
    \ce{I2Cl6 + 3H2O -> IO3- + ICl- + 4Cl- + 6H+}
\end{tightcenter}
\ce{I2Cl6}可以与芳基锡或芳基汞化合物作用生成二芳基碘鎓衍生物:
\begin{tightcenter}
    \ce{2PhSnCl3 + ICl3 -> Ph2ICl + 2SnCl4}
\end{tightcenter}
\subsubsection{\ce{XF5}和~\ce{IF7}的化学性质}
\paragraph{\ce{ClF5}和~\ce{BrF5}的化学性质}
\ce{ClF5}和~\ce{ClF3}一样具有强烈的氟化性,但有明确计量比的反应报道的比较少.~\ce{ClF5}与~\ce{H2O}剧烈反应生成~\ce{FClO2}:
\begin{tightcenter}
    \ce{ClF5 + 2H2O -> FClO2 + 4HF}
\end{tightcenter}
可见~\ce{ClF5}还不足以将~\ce{H2O}氧化,只是发生水解反应.~\ce{BrF5}也与水发生剧烈的水解反应:
\begin{tightcenter}
    \ce{BrF5 + 3H2O -> HBrO3 + 5HF}
\end{tightcenter}
\ce{BrF5}也是强的氟化剂,例如可以将硅酸盐完全氟化:
\begin{tightcenter}
    \ce{KAlSi3O8 + 8BrF5 ->[$\Delta$] KF + AlF3 + 3SiF4 + 4O2 + 8BrF3}
\end{tightcenter}
\indent ~\ce{ClF5}或~\ce{BrF5}与~\ce{AsF5},~\ce{SbF5}等强Lewis酸反应生成~\ce{[XF4]+[MF6]-},其中存在氟桥连接的长链结构,与\autoref{fig:[BrF4][Sb2F11]的结构}类似.稍弱的Lewis酸(包括~\ce{BF3},~\ce{PF5}和~\ce{TiF4}等)不能发生此类反应.\\
\indent ~\ce{ClF5}或~\ce{BrF5}也可以从氟离子供体(主要是碱金属氟化物)接受~\ce{F-}形成八面体型的~\ce{[XF6]-}.
\paragraph{\ce{IF5}和\ce{IF7}的化学性质}
相比~\ce{ClF5}和~\ce{BrF5},有关~\ce{IF5}的反应被更广泛和系统地研究过,主要是因为~\ce{IF5}的反应活性低得多.它与氧化物反应通常生成部分氟化的产物以及加合物,例如:
\begin{tightcenter}
    \ce{KMnO4 + IF5 -> MnO3F + IO3F + KF}\\
    \ce{V2O5 + 3IF5 -> 2VOF3.3IO3F}
\end{tightcenter}
尽管~\ce{IF5}反应性相对稍弱,但仍然在水中剧烈地水解:
\begin{tightcenter}
    \ce{IF5 + 3H2O -> HIO3 + 5HF}
\end{tightcenter}
\indent \ce{IF7}是比~\ce{IF5}更强的氟化剂,但其水解相对并不激烈:
\begin{tightcenter}
    \ce{IF7 + H2O -> IOF5 + 2HF}\\
    \ce{IOF5 + 3H2O -> HIO4 + 5HF}
\end{tightcenter}
\ce{IF7}脱去~\ce{F-}或得到~\ce{F-}分别生成的~\ce{[IF6]+}和~\ce{[IF8]-}离子都已经得到,其结构在前面已经介绍过.
\subsection{卤素阳离子}
\paragraph{\ce{X2+}离子}
人们很早就发现~\ce{I2}溶于发烟硫酸形成亮蓝色的顺磁性溶液.后来的研究表明~\ce{I2}被氧化为~\ce{I2+}离子.由此,人们又用强氧化剂制备了类似的亮红色阳离子~\ce{Br2+},而~\ce{Cl2+}目前只在低压放电管中观察到.典型的合成~\ce{Br2+}和~\ce{I2+}的方法是用~\ce{SbF5}氧化相应的单质\cite{Davies1974}\cite{Edwards1971}(制备\ce{Br2+}时需要加入定量的~\ce{BrF5}以避免~\ce{Br3+}等副产物的形成):
\begin{tightcenter}
    \ce{2I2 + 5SbF5 ->[SO2] 2[I2]+[Sb2F11]- + SbF3}\\
    \ce{9Br2 + 2BrF5 + 30SbF5 -> 10[Br2]+[Sb3F16]-}
\end{tightcenter}
后来\cite{Wilson1980}设计出了一条更简单的路线,即用~\ce{S2O6F2}氧化相应的单质,再用过量的~\ce{SbF5}溶剂解:
\begin{tightcenter}
    \ce{Br2 + S2O6F2 -> 2BrSO3F}\\
    \ce{Br2 + 2BrSO3F + 10SbF5 -> 2[Br2]+[Sb3F16]- + 2Sb2F9SO3F}
\end{tightcenter}
而~\ce{I2}被氧化的产物为~\ce{I_n(SO3F)}(其中$n=1,\,3,\,7$),后面的溶剂解过程类似.\\
\indent 分子轨道理论预测~\ce{X2+}的键级为$1.5$,实际测得的键长确实缩短,而力常数增大(见\autoref{table:卤素二聚阳离子以及对应卤素的部分结构参数});并且~\ce{X2+}阳离子是顺磁性的.

\paragraph{\ce{X3+}与\ce{X5+}离子}
三原子阳离子~\ce{I3+}和~\ce{Br3+}也可以由氧化剂直接氧化对应的卤素单质得到:
\begin{tightcenter}
    \ce{3Br2 + 2[O2]+[AsF6]- -> 2[Br3]+[AsF6]- + 2O2}\\
    \ce{3I2 + 3AsF5 ->[SO2] 2[I3]+[AsF6]- + AsF3}
\end{tightcenter}
和前面介绍的~\ce{XY2+}离子一样,~\ce{I3+}和~\ce{Br3+}的结构也是折线形的.在单质用量更高的情形下可以得到~\ce{X5+}离子,其具有中心对称的平面V型链状结构\cite{Apblett1986}:

\noindent 也可以将~\ce{X5+}离子看作是~\ce{X3+}离子与~\ce{X2}单质的加合物.
\paragraph{\ce{Cl}的多卤阳离子}
即使使用很强的氧化剂,例如~\ce{[O2]+},也无法得到游离的~\ce{Cl2+}阳离子.~\ce{Cl2}在~\ce{HF}中与~\ce{[O2]+[SbF6]-}反应生成紫色的~\ce{[Cl2O2]+}离子\cite{Drews1999},可以视作~\ce{Cl2+}与~\ce{O2}的电荷转移复合物.~\ce{[Cl2O2]+}的平面梯形结构如下:
\chemfig{[Cl2O2]+}{1}{\ce{[Cl2O2]+}的结构, $r(\ce{Cl-Cl})=\SI{191.6}{\pico\meter},\,r(\ce{Cl-O})=\SI{242.5}{\pico\meter},\,r(\ce{O-O})=\SI{118.5}{\pico\meter}$}
\noindent 对~\ce{[Cl2O2]+}的量子化学计算结果表明,其基态为$^2\text{A}_2$, SOMO轨道主要由~\ce{O2}单元的$\pi^\ast$轨道和~\ce{Cl2}单元的$\pi^\ast$轨道组成,未成对电子主要分布在~\ce{Cl}原子上.而第一激发态为$^2\text{B}_2$,主要是~\ce{O2}单元的电子向~\ce{Cl2}单元的$\pi^\ast$轨道发生电荷转移跃迁,正是这一跃迁导致~\ce{[Cl2O2]+}呈现明显的紫色\cite{Drews1999}.\\
\indent 使用~\ce{IrF6}氧化~\ce{Cl2}可以得到矩形的~\ce{[Cl4]+}离子\cite{Seidel2000}:
\begin{tightcenter}
    \ce{2Cl2 + IrF6 ->[HF] [Cl4]+[IrF6]-}
\end{tightcenter}

尽管氯的氧化物不算稳定,但是人们也对这类化合物进行了广泛的研究.

\subsubsection{制备与反应}
\paragraph{\ce{Cl2O}}
\ce{Cl2O}可以通过~\ce{Cl2}与新制~\ce{HgO}的反应制备:
\begin{tightcenter}
    \ce{2Cl2 + 2HgO -> HgCl2.HgO + Cl2O}
\end{tightcenter}
工业上大规模的制备也可以使~\ce{Cl2}与潮湿的~\ce{Na2CO3}反应:
\begin{tightcenter}
    \ce{2Cl2 + 2Na2CO3 + H2O -> 2NaHCO3 + 2NaCl + Cl2O}
\end{tightcenter}
\indent \ce{Cl2O}极易溶于水,并且与水反应生成~\ce{HOCl}:
\begin{tightcenter}
    \ce{Cl2O + H2O <=> 2HOCl}
\end{tightcenter}
因此~\ce{Cl2O}可以视作次氯酸酐.工业上~\ce{Cl2O}的主要用途是制备次氯酸盐,例如~\ce{Ca(ClO)2}等.\\
\indent \ce{Cl2O}也是较强的氧化剂,可以与~\ce{NH3}形成的气态混合物发生剧烈地爆炸:
\begin{tightcenter}
    \ce{3Cl2O + 10NH3 -> 2N2 + 6NH4Cl + 3H2O}
\end{tightcenter}
\paragraph{\ce{ClO2}}
\ce{ClO2}是最早发现的氯的氧化物,也是相对最稳定的氯的氧化物,在工业上大量地用于漂白纸浆.通常可以用各种还原剂还原~\ce{ClO3-}或~\ce{ClO2-}制备~\ce{ClO2}:
\begin{tightcenter}
    \ce{2NaClO3 + 2NaCl + 2H2SO4 -> 2ClO2 + Cl2 + 2Na2SO4 + 2H2O}\\
    \ce{5NaClO2 + 4HCl -> 4ClO2 + 5NaCl + 2H2O}\\
    \ce{2KClO3 + 2H2C2O4 -> K2C2O4 + 2ClO2 + 2CO2 + 2H2O}\\
    \ce{2AgClO3 + Cl2 -> 2ClO2 + O2 + 2AgCl}\\
    \ce{2NaClO2 + Cl2 -> 2ClO2 + 2NaCl}
\end{tightcenter}
需要注意的是用~\ce{Cl-}在酸性条件下还原~\ce{ClO3-}时,氧化产物为~\ce{Cl2}而非~\ce{ClO2}.这一过程的机理如下所示:
\begin{tightcenter}
    \ce{ClO3- + Cl- + H+ -> ClO2- + HOCl}\\
    \ce{ClO3- + ClO2- + 2H+ -> 2ClO2 + H2O}\\
    \ce{HOCl + Cl- + H+ -> Cl2 + H2O}
\end{tightcenter}
\indent \ce{ClO2}在水中的溶解度约为$\SI{8}{\gram\per\liter}$,溶液颜色为暗绿色.在光照下,中性水溶液中的~\ce{ClO2}迅速歧化为~\ce{HCl}和~\ce{HClO3}\cite{Bowen1932}:
\begin{tightcenter}
    \ce{2ClO2 + 3H2O ->[$h\nu$] HClO3 + HCl + O2}
\end{tightcenter}
反应的第一步是~\ce{ClO2}在光的作用下分解为~\ce{ClO}和~\ce{O}自由基,随后~\ce{ClO}与水反应生成~\ce{H2ClO2},而~\ce{H2ClO2}又与第二分子~\ce{ClO}生成~\ce{HCl}和~\ce{HClO3}.改变浓度和光的波长等条件,也可以得到更复杂计量比的产物\cite{Karpel1992}:
\begin{tightcenter}
    \ce{20ClO2 + 10H2O ->[$h\nu$] 8HCl + 12HClO3 + 7O2}
\end{tightcenter}
\ce{ClO2}在碱性溶液中歧化为~\ce{ClO2-}和~\ce{ClO3-};同时可能发生副反应生成~\ce{O2}:
\begin{tightcenter}
    \ce{2ClO2 + 2OH- -> ClO2- + ClO3- + H2O}\\
    \ce{4ClO2 + 4OH- -> 4ClO2- + 2H2O + O2}
\end{tightcenter}
相关的反应机理比较复杂,具体可以参考\cite{Odeh2002}.\\
\indent 气相中的~\ce{ClO2}在光照下形成复杂的混合物\cite{Lopez1994},包含~\ce{Cl2},~\ce{O2}和少量~\ce{ClO3},后者二聚为~\ce{Cl2O6}或进一步光解为~\ce{ClO2}.固相中的~\ce{ClO2}光解生成~\ce{Cl2O3}和~\ce{Cl2O6}.
\paragraph{\ce{Cl2O4}}
需要注意的是,这里所说的~\ce{Cl2O4}(即~\ce{ClOClO3})并非~\ce{ClO2}的二聚体,最好视作高氯酸氯或者高氯酸和次氯酸的混合酸酐.~\ce{Cl2O4}可以由下面的反应制备:
\begin{tightcenter}
    \ce{CsClO4 + ClOSO2F ->[$\SI{-45}{\celsius}$] CsSO3F + ClOClO3}
\end{tightcenter}
\indent \ce{Cl2O4}非常不稳定,室温时分解为~\ce{Cl2},~\ce{O2}和~\ce{Cl2O6}.对其化学性质的研究相对较少.
\paragraph{\ce{Cl2O6}}
尽管前述一些物质的分解可以生成~\ce{Cl2O6},但是最好由~\ce{ClO2}与~\ce{O3}的反应制备:
\begin{tightcenter}
    \ce{2ClO2 + 2O3 -> Cl2O6 + 2O2}
\end{tightcenter}
\indent \ce{Cl2O6}与水反应生成~\ce{HClO3}和~\ce{HClO4}:
\begin{tightcenter}
    \ce{Cl2O6 + 2H2O -> HClO3 + HClO4}
\end{tightcenter}
因此可以将其视作氯酸和高氯酸的混合酸酐.前面已经介绍过~\ce{Cl2O6}的固相实际上是~\ce{[ClO2]+[ClO4]-}离子晶体,在某些反应中也可以体现这一性质:
\begin{tightcenter}
    \ce{Cl2O6 + HF -> FClO2 + HClO4}
\end{tightcenter}
\paragraph{\ce{Cl2O7}}
\ce{Cl2O7}是高氯酸的酸酐,可以用~\ce{P2O5}对~\ce{HClO4}小心地脱水制备得到:
\begin{tightcenter}
    \ce{4HClO4 + P4O10 ->[$\SI{-10}{\celsius}$] 2Cl2O7 + 2H2P2O6}
\end{tightcenter}
\indent \ce{Cl2O7}是油状液体,具有强氧化性,在振动下容易爆炸.相比低价的氯氧化物而言,~\ce{Cl2O7}的化学性质相对温和一些,不会点燃有机物质,在水中或碱性溶液中水解为高氯酸或高氯酸盐.
\subsection{\ce{Br}和~\ce{I}的氧化物}
\ce{Br}和~\ce{I}的稳定氧化物种类少于~\ce{Cl}的氧化物,目前有详细研究的只有~\ce{Br2O3},~\ce{Br2O5},~\ce{BrO2},~\ce{I2O5}和~\ce{I4O9}等少数几种,并且对Br的氧化物的了解远远少于氯和碘的氧化物.这里分别介绍它们的性质.
\subsubsection{\ce{Br}的氧化物}
\ce{Br2O3}和~\ce{Br2O5}两种氧化物都可以由~\ce{Br2}与~\ce{O3}在低温下反应得到,分别是橙黄色和无色的.它们的结构都类似于对应的~\ce{Cl}化合物,但对化学性质的研究仍然较少.
\subsubsection{\ce{I}的氧化物}
\ce{I}的氧化物是相对而言最稳定的卤素氧化物.
\paragraph{\ce{I2O5}}
早在1813年Gay Lussac研究~\ce{I2}的性质时就制备得到了~\ce{I2O5}.现在合成~\ce{I2O5}通常是在加热\ce{HIO3}脱水得到.\\
\indent \ce{I2O5}形成白色且稳定的吸湿性晶体,其中存在五配位的~\ce{I}共用~\ce{O}原子形成的长链,如下图所示:
\chemfig{I2O5-crys}{0.125}{\ce{I2O5}的晶体结构\quad$\text{P}\,2\,1/\text{c}$\quad$a=\SI{1103.6}{\pico\meter},b=\SI{506.3}{\pico\meter},c=\SI{813.5}{\pico\meter},\beta=107.18^\circ$}
\ce{I2O5}对水的结合能力很强,市售的~\ce{I2O5}通常是~\ce{HI3O8}(即~\ce{I2O5.HIO3}).~\ce{I2O5}也容易溶于水形成~\ce{HIO3}.\\
\indent \ce{I2O5}能快速地将~\ce{CO}氧化为~\ce{CO2}:
\begin{tightcenter}
    \ce{I2O5 + 5CO -> I2 + 5CO2}
\end{tightcenter}
这一反应常常被用于测定混合气体中~\ce{CO}的含量.\\
\paragraph{\ce{I4O9}}
\indent \ce{I2}与~\ce{O3}反应可以得到~\ce{I4O9}.加热~\ce{HIO3}和~\ce{H3PO4}的混合物也可以得到~\ce{I4O9}:
\begin{tightcenter}
    \ce{4I2 + 9O3 -> I4O9 + 9O2}\\
    \ce{8HIO3 -> 2I4O9 + 4H2O + O2}
\end{tightcenter}
\ce{I4O9}中含有$+3$价和$+5$价的碘,也许将其组成视作~\ce{I(IO3)3}比较好.目前尚没有对~\ce{I4O9}的结构的研究.\ce{I4O9}受热至$\SI{75}{\celsius}$以上分解为~\ce{I2O5}和~\ce{I2}:
\begin{tightcenter}
    \ce{4I4O9 ->[$\Delta$] 6I2O5 + 2I2 + 3O2}
\end{tightcenter}
\ce{I4O9}也可以与水反应生成~\ce{HIO3}和~\ce{I2}:
\begin{tightcenter}
    \ce{5I4O9 + 9H2O -> 18HIO3 + I2}
\end{tightcenter}
\paragraph{\ce{I2O4}}
将~\ce{HIO3}与浓~\ce{H2SO4}混合并长时间加热即可得到柠檬黄色的~\ce{I2O4}:
\begin{tightcenter}
    3HIO3 -> I2O4 + HIO4 + H2O
\end{tightcenter}
也可以由~\ce{(IO)2SO4}的水解得到\cite{Daehlie1964}:
\begin{tightcenter}
    \ce{4(IO)2SO4 + 4H2O -> 3I2O4 + I2 + 4H2SO4}
\end{tightcenter}
\indent \ce{I2O4}固体中存在四配位的~\ce{I^{V}}与二配位的~\ce{I^{III}}交替连接形成的链状结构,如下图所示\cite{Fjellvaag1994}:

\subsubsection{\ce{Cl}的氟氧化物}
\paragraph{\ce{FClO}}
首次得到~\ce{FClO}是在研究~\ce{ClF3}与~\ce{H2O}反应的过程中:
\begin{tightcenter}
    \ce{ClF3 + H2O -> FClO + 2HF}
\end{tightcenter}
\ce{FClO}是不稳定的化合物,在室温下以很快的速度分解:
\begin{tightcenter}
    \ce{2FClO ->[$\SI{25}{\celsius}$] FClO2 + ClF}\quad$t_{1/2}=\SI{25}{\second}$
\end{tightcenter}
\paragraph{\ce{FClO2}}
\ce{FClO2}可以由~\ce{ClO2}与~\ce{F2}在低温下反应得到:
\begin{tightcenter}
    \ce{ClO2 + F2 -> 2FClO2}
\end{tightcenter}
也可以由~\ce{NaClO3}与~\ce{ClF3}反应得到:
\begin{tightcenter}
    \ce{6NaClO3 + 4ClF3 -> 6FClO2 + 2Cl2 + 3O2 + 6NaF}
\end{tightcenter}
相比~\ce{FClO},~\ce{FClO2}的稳定性更高一些,加热至$\SI{300}{\celsius}$时其分解为~\ce{ClF}和\ce{O2}:
\begin{tightcenter}
    \ce{2FClO2 ->[$\SI{300}{\celsius}$] 2ClF + 2O2}
\end{tightcenter}
尽管~\ce{FClO2}中~\ce{Cl}的氧化态低于~\ce{FClO3},但它的反应活性远远高于~\ce{FClO3}.这可能是由于~\ce{FClO2}中的~\ce{Cl}没有达到配位饱和,具有更强的亲电性.~\ce{FClO2}在水中缓慢水解分解为~\ce{HF}和~\ce{ClO2},而在碱性水溶液中则发生简单的水解生成~\ce{F-}和~\ce{ClO3-}:
\begin{tightcenter}
    \ce{4FClO2 + 2H2O -> 4HF + 2ClO2 + O2}\\
    \ce{2FClO2 + 2OH- -> F- + ClO3- + H2O}
\end{tightcenter}
\indent \ce{FClO2}也是优良的氟化试剂.它既可以作为~\ce{F-}的供体,也可以作为其受体:
\begin{tightcenter}
    \ce{FClO2 + SbF5 -> [FClO2]+[SbF6]-}\\
    \ce{FClO2 + KF -> K+[F2ClO2]-}
\end{tightcenter}
\paragraph{\ce{FClO3}}
\ce{FClO3}是氯的氟氧化物中最稳定的.它可以由~\ce{KClO4}的氟化或用~\ce{F2}等氟化试剂处理~\ce{KClO3}得到:
\begin{tightcenter}
    \ce{KClO4 + 2HF + SbF5 -> FClO3 + KSbF6 + H2O}\\
    \ce{KClO3 + F2 -> FClO3 + KF}
\end{tightcenter}
室温下~\ce{FClO3}的反应活性很低(主要是由于其四面体结构而具有很高的动力学稳定性),在高温下才表现出的明显的氧化性,并且具有很高的比冲,因此被广泛地用于火箭推进剂中地氧化剂.同时,~\ce{FClO3}也具有很高的阻抗,因此可以被用于高压系统中的绝缘气体.\\
\indent \ce{FClO3}的水解非常缓慢,在高压密封管中加热至$\SI{300}{\celsius}$才能被浓~\ce{NaOH}溶液完全水解:
\begin{tightcenter}
    \ce{FClO3 + 2NaOH -> NaClO4 + NaF + H2O}
\end{tightcenter}
换用更强的碱,例如醇钾或氨基钠,则可以在室温下反应:
\begin{tightcenter}
    \ce{FClO3 + ROK -> ROClO3 + KF}\\
    \ce{FClO3 + 2NaNH2 -> Na+[HNClO3]- + NaF + NH3}
\end{tightcenter}
同时,~\ce{FClO3}也几乎不作为~\ce{F-}的供体或受体.\\
\indent \ce{FClO3}与有机物反应生成复杂的产物:
\begin{tightcenter}
    \ce{FClO3 + LiPh -> PhClO3 + LiF}\\
    \ce{FClO3 + Li[C6H4S] -> C6H4SF + LiClO3}
\end{tightcenter}
推测后一个反应是生成~\ce{-ClO3}取代的噻吩后~\ce{F-}再发生芳香亲核取代反应将~\ce{-ClO3}基团转化为~\ce{-F}.
\subsubsection{\ce{Br}的氟氧化物}
相比~\ce{Cl}的氟氧化物,~\ce{Br}的氟氧化物的稳定性更差而反应性更高,相关的研究也相对较少.
\paragraph{\ce{FBrO2}}
\ce{FBrO2}是不稳定的无色液体,可由下面的方法制备:
\begin{tightcenter}
    \ce{KBrO3 + BrF5 -> FBrO2 + K[F4BrO]}\\
    \ce{K[F4BrO] + KBrO3 -> 2K[F2BrO2]}\\
    \ce{K[F2BrO2] + HF -> KHF2 + FBrO2}
\end{tightcenter}
\ce{FBrO2}水解也产生卤酸根离子,但反应猛烈以致爆炸.在$\SI{0}{\celsius}$时\ce{FBrO2}在碱性溶液中水解为~\ce{BrO3-}和~\ce{F-}.
\paragraph{\ce{FBrO3}}
\ce{FBrO3}相比~\ce{FClO3}更加活泼,在室温下与水,碱溶液甚至玻璃发生反应:
\begin{tightcenter}
    \ce{FBrO3 + H2O -> BrO4- + HF + H+}\\
    \ce{FBrO3 + 2OH- -> BrO4- + F- + H2O}
\end{tightcenter}
\subsubsection{\ce{I}的氟氧化物}
与~\ce{Br}和~\ce{Cl}不同,~\ce{I}的氟氧化物只有$+5$价与$+7$价,而没有~\ce{FIO}.相关物质的稳定性高于~\ce{Br}的类似化合物.
\paragraph{\ce{FIO2}}
室温时在无水~\ce{HF}中对~\ce{I2O5}的直接氟化或加热~\ce{F3IO}使其分解可以制备~\ce{FIO2}:
\begin{tightcenter}
    \ce{2I2O5 + 2F2 -> 4FIO2 + O2}\\
    \ce{2F3IO ->[$\SI{110}{\celsius}$] FIO2 + IF5}
\end{tightcenter}

\paragraph{\ce{F3IO}}
将~\ce{I2O5}溶解在沸腾的~\ce{IF5}中,然后冷却混合物即可得到无色晶体~\ce{F3IO}:
\begin{tightcenter}
    \ce{I2O5 + 3IF5 ->[$\SI{100}{\celsius}$] 5F3IO}
\end{tightcenter}
正如前面所提到的,~\ce{F3IO}在加热时分解为~\ce{FIO2}和~\ce{IF5}.
\paragraph{\ce{FIO3}}
将~\ce{KIO4}在无水~\ce{HF}中溶剂解可以得到无色晶体~\ce{FIO3}:
\begin{tightcenter}
    \ce{KIO4 + 2HF -> FIO3 + KF + H2O}
\end{tightcenter}
鉴于~\ce{I}较大的半径,四面体构型的~\ce{FIO3}不算稳定,可以与~\ce{BF3}或~\ce{AsF5}等Lewis酸反应形成~\ce{[IO3]+}.
\paragraph{\ce{F3IO2}}
用氟磺酸对高碘酸盐的部分氟化可以得到黄色固体~\ce{F3IO2}:
\begin{tightcenter}
    \ce{Ba3H4(IO6)2 + 8HSO3F -> 2HIO2F4 + 3BaSO4 + 5H2SO4}\\
    \ce{HIO2F4 + SO3 -> F3IO2 + HSO3F}
\end{tightcenter}

\section{VIIA族元素的含氧酸及含氧酸盐}
\subsection{\ce{F}的含氧酸}
氟在卤素中是独特的,只能形成氧化态为$-1$的化合物.因此已知的~\ce{F}的含氧酸只有次氟酸~\ce{HOF},并且也没有已知的次氟酸盐.有关~\ce{HOF}的相关性质已经在~\ce{O}的氟化物的部分介绍过,因此这里不再赘述.
\subsection{\ce{Cl},~\ce{Br}和~\ce{I}的含氧酸}
\subsubsection{次卤酸~\ce{HOX}}
尚未分离出纯净的次卤酸~\ce{HOCl},~\ce{HOBr}和~\ce{HOI},但是它们的水溶液及盐的性质已经得到充分的研究.各卤素单质在冷的碱溶液中迅速歧化生成相应的次卤酸根离子~\ce{XO-}和卤素离子~\ce{X-}:
\begin{tightcenter}
    \ce{X2 + 2OH- -> XO- + X- + H2O}
\end{tightcenter}
然而次卤酸及其阴离子都是不稳定的,会发生歧化.在室温下,~\ce{ClO-}的歧化很慢,而~\ce{BrO-}和~\ce{IO-}的歧化则非常快:
\begin{tightcenter}
    \ce{3XO- -> XO3- + 2X-}
\end{tightcenter}
因此一般条件下~\ce{Br2}和~\ce{I2}单质在碱性溶液中歧化实际上直接生成~\ce{XO3-}和~\ce{X-}.在一定条件下(例如~\ce{Co^{II}}的催化),次卤酸根也可以按照如下的方式分解:
\begin{tightcenter}
    \ce{2XO- -> 2X- + O2}
\end{tightcenter}
相比而言,次卤酸的稳定性弱于次卤酸根的稳定性.
\paragraph{\ce{HOCl}}
人们了解和使用次氯酸的历史由来已久,主要由于次氯酸及其盐的漂白和消毒作用而在工业和家用方面有广泛的应用(例如,常用的84消毒液\footnote{由北京地坛医院在1984年研制,最初名为84肝炎洗消液,经鉴定获北京市科技成果奖,后更名为84消毒液.}的主要成分是~\ce{NaClO}).工业上常用各种碱与~\ce{Cl2}的反应制备次氯酸盐,例如用~\ce{NaOH}与~\ce{Cl2}反应制备~\ce{NaClO},以及用~\ce{CaO}与~\ce{Cl2}反应制备~\ce{Ca(ClO)2}(漂白粉的主要成分):
\begin{tightcenter}
    \ce{Cl2 + 2NaOH -> NaCl + NaClO + H2O}\\
    \ce{2Cl2 + 2CaO -> CaCl2 + Ca(ClO)2}
\end{tightcenter}
如果要制备不含~\ce{Cl-}的产物,可以采用~\ce{Cl2O}水解的方式.事实上,高浓度的~\ce{HOCl}水溶液呈现黄色的部分原因是由于其中存在相当量的~\ce{Cl2O}.\\
\indent \ce{HOCl}和~\ce{ClO-}是常见氧化剂中氧化性较强的,其反应机理通常是~\ce{O}原子的直接迁移,因此还原产物通常是~\ce{Cl-}离子.~\ce{ClO-}可以将~\ce{IO3-}氧化为~\ce{IO4-},将~\ce{Cr^{3+}}氧化为~\ce{[CrO4]^{2-}},在碱性条件下将~\ce{Fe^{3+}}氧化为~\ce{[FeO4]^{2-}}:
\begin{tightcenter}
    \ce{ClO- + IO3- -> Cl- + IO4-}\\
    \ce{3ClO- + 2M^{3+} + 4OH- -> 3Cl- + 2[MO4]^{2-} + 2H2O}\quad\ce{M = Cr, Fe}
\end{tightcenter}
使用次氯酸盐消毒的水通常有特征的刺激性气味,这实际上是由于与水中的胺类物质反应生成氯胺:
\begin{tightcenter}
    \ce{HOCl + R2NH -> R2NCl + H2O}
\end{tightcenter}
\paragraph{\ce{HOBr}和~\ce{HOI}}
\ce{HOBr}和~\ce{HOI}的性质与~\ce{HOCl}类似,但它们的稳定性更差.通过将~\ce{Br2}加入冷的浓碱溶液中可以制得~\ce{NaBrO.5H2O}和~\ce{KBrO.3H2O}等水合物,但尚未值得次碘酸盐.\\
\indent \ce{BrO-}与胺反应定量地生成~\ce{N2},这一反应可以用于定量分析尿素:
\begin{tightcenter}
    \ce{(NH2)2CO + 3BrO- + 2OH- -> N2 + 3Br- + 3H2O + CO3^{2-}}
\end{tightcenter}
\indent 次卤酸盐的一个典型反应是卤仿反应,即在碱性条件下与甲基酮反应生成卤仿~\ce{CHX3}:
\begin{tightcenter}
    \ce{RCOCH3 + 3XO- -> CHX3 + RCOO- + 2OH-}
\end{tightcenter}
因此,也可以用对应的单质在碱性条件下进行此反应.由于碘仿~\ce{CHI3}是具有特殊气味的黄色固体,因此常用此方法检测甲酰基.需要注意,~\ce{-CH(OH)CH3}基团在此条件下也可被氧化为~\ce{-COCH3}进而发生反应,因此诸如乙醇等也可以被该反应检测.
\subsubsection{亚卤酸~\ce{HXO2}}
亚卤酸~\ce{HXO2}(按实际结构写作~\ce{HOXO}更好)是卤素的含氧酸中最不稳定的.亚氯酸~\ce{HClO2}和~\ce{HClO}一样不存在游离态,并且只能在较稀的水溶液中存在;亚氯酸盐则稳定得多,例如~\ce{NaClO2}已经被广泛生产与使用.亚溴酸~\ce{HBrO2}和亚碘酸~\ce{HIO2}则更不稳定,几乎只能以反应中间体的形式存在,也没有得到亚碘酸盐.\\
\indent 我们在介绍~\ce{ClO2}时已经说过其在碱性溶液中歧化为~\ce{ClO2-}和~\ce{ClO3-},然后酸化即可得到~\ce{HClO2}水溶液.更好的制备方法是用稀~\ce{H2SO4}与~\ce{Ba(ClO2)2}反应制备:
\begin{tightcenter}
    \ce{Ba(OH)2 + H2O2 + 2ClO2 -> Ba(ClO2)2 + 2H2O + O2}\\
    \ce{Ba(ClO2)2 + H2SO4 -> BaSO4 + 2HClO2}
\end{tightcenter}
\indent 亚氯酸的分解方式比较复杂,并且对浓度, pH值等条件比较敏感.分解的主要形式是生成~\ce{ClO2}:
\begin{tightcenter}
    \ce{5HClO2 -> 4ClO2 + HCl + 2H2O}
\end{tightcenter}
副反应可能生成~\ce{ClO3-}或~\ce{O2}.\\
\indent 制备亚氯酸盐通常在碱性条件下用~\ce{H2O2}还原~\ce{ClO2},这样可以不引入额外杂质(参考前面制备~\ce{Ba(ClO2)2}的反应).~\ce{ClO2-}的碱性溶液是稳定的,但在加热或光照时发生分解:
\begin{tightcenter}
    \ce{3ClO2- ->[\text{碱性溶液}] 2ClO3- + Cl-}\\
    \ce{6ClO2- ->[$h\nu$][$\text{pH}=8.4$] 2ClO3- + 4Cl- + 3O2}\\
    \ce{10ClO2- ->[$h\nu$][$\text{pH}=4.0$] 2ClO3- + 6Cl- + 2ClO4- + 3O2}\\
\end{tightcenter}
在所有亚氯酸盐中最主要的是~\ce{NaClO2},工业上用于纺织品的漂白等方面.
\subsubsection{卤酸~\ce{HXO3}}
卤酸~\ce{HXO3}的稳定性较亚卤酸~\ce{HXO2}更高.尽管~\ce{HClO3}和~\ce{HBrO3}目前仍然没有得到纯净的游离态,但它们的盐已经得到充分的研究.~\ce{HIO3}则相对稳定,在室温下是白色固体.\\
\indent 可以用卤素单质~\ce{X2}在热的碱溶液中的歧化反应制备~\ce{XO3-}(注意对比在冷的碱溶液中产物的差异):
\begin{tightcenter}
    \ce{3X2 + 6OH- -> 5X- + XO3- + 3H2O}
\end{tightcenter}
现在也更常用电解法制备~\ce{ClO3-}.工业上主要制备~\ce{NaClO3}并进一步用于生产~\ce{ClO2},以及~\ce{KClO3}用于焰火和火药等.\\
\indent 溴酸盐和碘酸盐的制备规模和用途少得多,通常使用化学氧化的方式,例如
\begin{tightcenter}
    \ce{Br- + 3Cl2 + 6OH- -> 3H2O + BrO3- + 6Cl-}\\
    \ce{I2 + 2NaClO3 -> 2NaIO3 + Cl2}
\end{tightcenter}
后者能发生是因为碘酸盐的热稳定性更强.
